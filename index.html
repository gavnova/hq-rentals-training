💡 Pro Tips
Things experienced staff wish they knew sooner
Vehicle Class vs Vehicle: Class = the model type. Vehicle = the physical car. Each car needs its own record.
Set to Reserved first: New cars should be "Reserved" status until fully onboarded — prevents accidental bookings.
